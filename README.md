# Secure OTP Generator

A modern, secure, and interactive **One-Time Password (OTP) generator** built with **HTML, CSS, and JavaScript**.  
Generates **6-digit numeric OTPs** securely in the browser and allows instant verification.

---

## Features

- Generate **secure 6-digit OTPs** using the Web Crypto API.
- Verify OTPs instantly in the UI.
- Modern and responsive design.
- Interactive feedback: ✅ success and ❌ error messages.
- Easy to integrate into your login, signup, or verification system.
